Problem 4.1 (25 pts)

In the beaded string shown in Figure 1, the interval between neighboring beads is a, and the distance from the end beads to the wall is a/2. All the beads have mass m and are constrained to move only vertically in the plane of the paper. The strings are massless with constant string tension T.

[Figure 1: Beads on a string — beads numbered 1, 2, 3, ..., N−1, N spaced a apart, with the end beads a distance a/2 from each fixed wall.]

(a) Show that the physics of the left-hand wall can be incorporated by going to an infinite system and requiring the boundary condition A1 = −A0.

Problem 4.2 (25 pts)

A physicist was trying to understand a system with two coupled torsional pendula. First, she tried to write down the equations of motion of the pendula in terms of θ1 and θ2, which are the angles with respect to the equilibrium positions of the pendula. She found that the interaction matrix M^(−1)K commutes with the 2 × 2 reflection symmetry matrix
    S = [ 0  −1 ]
        [ −1  0 ]
(i.e., satisfies the condition [S, M^(−1)K] = 0).

(a) How many normal modes do we have in this system?
(b) What will be the amplitude ratio (A1/A2, where A1 and A2 are the components of the eigenvector) of the torsional pendula in each normal mode?

Problem 4.4 (25 pts)

A string of mass density µ and stretched with tension T is deformed as shown in Figure 3 (amplitude A is very small; the deformation as shown in the figure is greatly exaggerated). The string is released at t = 0 with zero initial velocity (∂y(x,t)/∂t = 0 at t = 0 for all x).

[Figure 3: String deformation — a triangular shape on the interval [−L, L], equal to 0 at x = −L, rising to +A at x = −L/2, passing through 0 at x = 0, dropping to −A at x = L/2, and returning to 0 at x = L.]

(a) How many normal modes do we have in this system?
(b) Find expressions for the Fourier coefficients Am of the normal mode expansion of the initial string shape and use them to write a full time-dependent series that describes the motion of the string at t > 0: y(x,t).
(c) Make a sketch of the amplitudes Am of the modes as a function of mode number m. Make sure that you indicate amplitudes for all possible normal modes of the string.
