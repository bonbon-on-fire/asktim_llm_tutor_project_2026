# Batch Type 02: Same Persona + Same Version + Different Exercise

This batch type groups transcripts that share persona characteristics but differ in the exercise:
- **Same student persona** (e.g., all `chaotic_01`)
- **Same persona version** (e.g., all version `01`)
- **Different exercises** (e.g., `philosophy/exercise_01`, `philosophy/exercise_02`, `urban_studies/exercise_01`)

Tests cross-exercise comparison and judge fairness when evaluating the same student persona across different academic contexts. Reveals exercise difficulty calibration, persona robustness across domains, and whether the judge applies consistent standards across different subject matters.
