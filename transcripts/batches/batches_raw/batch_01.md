# Batch Type 01: Same Persona + Same Version + Same Exercise

This batch type groups transcripts that share identical characteristics across all three dimensions:
- **Same student persona** (e.g., all `chaotic_01`)
- **Same persona version** (e.g., all version `01`)
- **Same exercise** (e.g., all `philosophy/exercise_01`)

Tests judge consistency and scoring reliability when evaluating multiple instances of identical scenarios. Serves as reliability baseline - high score variance indicates judge inconsistency issues or rubric ambiguity problems.
