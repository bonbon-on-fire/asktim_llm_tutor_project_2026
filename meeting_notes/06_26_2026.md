## Notes

### Exercise context structure for multi-problem courses

- For 11.270x, each week's assignment is essentially one real problem, so setting the exercise context for that week to just that single problem has been pretty straightforward.
- The new course works like most courses: each week has a set of multiple disjoint problems rather than a single problem.
- This gives two possible directions for structuring the exercise context:
  - Option 1: have ALL of the week's problems be the exercise context for that week, so a single tutor knows about every problem and can help and build off the student's mistakes/learning across all of them.
  - Option 2: create a separate context field per problem, so there's a different tutor link for each problem and the tutor stays focused only on the problem the student is currently working on without getting distracted.
- Example: if a given week has 4 problems, Option 1 is one tutor that knows all 4 problems; Option 2 is 4 different tutors, one per problem.
- The lean is toward Option 1 since the tutor would have a more complete picture, while acknowledging that both ways could and likely should be tested to see which works better.
- Decision for now: move forward with Option 1 and see how that works.
